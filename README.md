## 💻 Sobre o Projeto

Este projeto foi desenvolvido para servir como um acervo vivo das minhas habilidades em desenvolvimento web e engenharia de software. A interface foi construída em cima de uma forte identidade visual "Cyber/Glassmorphism", unindo alto desempenho de leitura, responsividade e interatividade de interface.

> **Objetivo:** Facilitar a análise do meu perfil profissional por recrutadores e parceiros através de uma experiência de usuário (UX) premium, fluida e não-disruptiva.

---

## ✨ Funcionalidades Principais

- **Visualizadores Modais Integrados**: Ao clicar em projetos ou certificados, as informações (como mockups de repositórios ou leitores simulados de PDFs) abrem em pop-ups dinâmicos, evitando que o usuário saia incorretamente da página principal.
- **Glassmorphism & UI Suave**: Elementos flutuantes, bordas arredondadas harmônicas e sombras translúcidas.
- **Micro-interações & Background Ativo**: Motor em Vanilla JS controlando um visualizador de partículas em tempo real integrado simetricamente ao rastreamento de cursor do usuário.
- **Motor de Renderização Responsivo**: O layout se ajusta completamente do Desktop puro (Grids de 3 colunas) às telas de dispositivos móveis usando Fallbacks otimizados em Flexbox.
- **Padrão Semântico & Vetorial**: Ícones de tecnologias e logos minuciosamente redesenhados do zero utilizando SVGs inline puramente desenhados por `paths`, garantindo nitidez absoluta na interface e nenhuma perda no tempo de carregamento de recursos visuais externos.

---

## 📂 Arquitetura do Repositório

O projeto é mantido puramente via tecnologias padrão da Web (Vanilla), garantindo tempo de carregamento quase nulo, zero peso morto e total estabilidade de *build* na hospedagem nativa do GitHub.

```text
/
├── index.html       # Arquitetura estrutural e metadados de marcação de atributos (DOM)
├── style.css        # Conjunto de regras globais, variáveis base, Design System e Grids
├── script.js        # Lógica assíncrona, renderizador de interações e injetor de Modais
└── imgs/            # Recursos visuais independentes usados pelas telas de ambiente
```

## ⚡ Formatos de Hospedagem

A interface já está homologada para instâncias rápidas baseadas no ecossistema e ambiente corporativo do GitHub Pages:

1. Faça o **Fork** do repositório ou clone na sua máquina.
2. Acesse as **Settings (Configurações)** do Repositório.
3. No painel principal, acesse a divisão lateral **Pages**.
4. Em **Build and Deployment**, defina o apontamento para o branch `/ (root) -> main`.
5. Em alguns instantes, todo o acervo portfólio/currículo digital estará no ar sem burocracia CI/CD.
