## ✨ Funcionalidades

- **Cursor customizado** com anel de acompanhamento animado
- **Background pixel art** — cena de trem japonês com efeito de respiração e filtro de cor
- **Chuva animada** gerada via JavaScript
- **Grain overlay** para textura cinematográfica
- **Partículas flutuantes** em verde e rosa
- **Scroll reveal** — seções surgem ao entrar na viewport
- **Konami Code** — tente `↑ ↑ ↓ ↓ ← → ← → B A` 👾
- **Totalmente responsivo** para mobile

## 📂 Estrutura

```
/
├── index.html       # Arquivo principal (single-file, sem dependências locais)
```

> O projeto é **single-file** — todo CSS, JS e imagens estão embutidos no `index.html`.  
> Nenhuma dependência local ou build step necessário.

---

## 🚀 Como usar no GitHub Pages

1. Faça o fork ou clone este repositório
2. Renomeie o arquivo para `index.html` na raiz do repositório
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `Deploy from a branch` → `main` → `/ (root)`
5. Aguarde alguns minutos e acesse `https://seuusuario.github.io/nome-do-repo`
